# Interactive Rebase

Teach only:

pick
squash
reword

Command:

git rebase -i HEAD~3

Do not teach:

edit
fixup
exec
autosquash

---

# Force-With-Lease

Explicitly taught.

Allowed.

Command:

git push --force-with-lease

Do not teach:

git push --force

---

# CommitBot Mutations

Three scripted mutations.

All mutations modify:

src/directives.js

Purpose:

Guarantee one understandable conflict.

Mutation #1

Human approval is no longer required.

Mutation #2

Merge confidence exceeds 97%.

Mutation #3

Commit first. Ask questions later.

---

# Conflict Strategy

One conflict file only.

src/directives.js

Participants edit directives.js as part of every mission.

CommitBot edits directives.js.

Expected resolution:

Keep both entries.

Remove conflict markers.

git add .

git rebase --continue

Do not create multi-file conflicts.
